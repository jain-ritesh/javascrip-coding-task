# 🚀 10 JavaScript Coding Questions

This repository contains solutions to **10 basic-to-intermediate JavaScript coding problems**.  
These problems are focused on strengthening core concepts like loops, arrays, conditionals, and problem-solving logic.

---

## 📌 Problem List

### ✅ Q1: Even Counter
Count how many numbers in an array are even using a loop.

---

### ✅ Q2: Sum of Prices
Calculate the total sum of multiple product prices using a loop.

---

### ✅ Q3: Password Attempts
Allow a user **3 attempts** to enter the correct password using a loop.

---

### ✅ Q4: Print Pattern
Print a **star (*) pattern of 5 rows** using loops.


---

### ✅ Q5: Reverse Array
Reverse an array using a loop **without using built-in methods** like `.reverse()`.

---

### ✅ Q6: Find Maximum Number
Find the **largest number** in an array using a loop.

---

### ✅ Q7: Shopping Cart with Discount
Apply discount to items above a certain price and calculate the **final total**.

---

### ✅ Q8: Student Marks Average
Calculate the **average marks** from an array of student scores.

---

### ✅ Q9: Frequency Counter
Count how many times each number appears in an array.

### ✅ Q10: Order Billing System
Create a loop-based system that:
- Displays a menu
- Takes item quantity
- Calculates total bill
- Adds GST
